# NDCM Math Club Website

Official website for the **Notre Dame College Mymensingh Math Club**.

## Features

- **Bilingual Support**: Toggle between English and Bengali (using Noto Serif Bengali).
- **Mathematical Rendering**: Full LaTeX support via KaTeX.
- **Interactive Visualizations**: Dynamic math graphs on the landing page.
- **Single Page Application**: Fast navigation without page reloads.

## Technologies

- HTML5 / CSS3 / JavaScript (ES6+)
- [Tailwind CSS](https://tailwindcss.com) (CDN)
- [KaTeX](https://katex.org) for math rendering
- Hosted on GitHub Pages

## SEO

The site includes:
- `sitemap.xml` for search engine indexing.
- `robots.txt` for crawler instructions.
- JSON-LD Structured Data for Organization recognition.
- Open Graph tags for social media sharing.
